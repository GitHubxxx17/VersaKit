VersaKit 是前端高质量的 React 组件库，帮助开发者快速产出高质量产品原型。

## 📦 Install

```bash
npm install versakit
```

```bash
yarn add versakit
```

```bash
pnpm add versakit
```

## 🔨 Usage

```tsx
import { Button, DatePicker } from "versakit";

export default () => (
  <>
    <Button type="primary">PRESS ME</Button>
    <DatePicker placeholder="select date" />
  </>
);
```
